My goal is to be able to instruct future readers to reproduce the same project based on my build instructions.

During this period I was made aware that I need to purchase a Bi-Directional logic converter to convert my sensor’s voltage to read on the Pi’s GPio pins. The TPA81 sensor takes a voltage of 5V and the Pi’s I2C pins take a capacity of 3.3V. As a results, the Bi-Directional converter will convert the voltage and vice versa so the Raspberry Pi would not be destroyed. This was a problem encountered and financial update to my project as I had to spend an additional $7.33. Without the Bi-Directional converter this project would be unsuccessful as I would not be able to convert a higher voltage level to a lower one. Thus, I would not able to take reading from my sensor which is why this part was extremely significant. Another problem was when I finished assembling my project the Raspberry Pi did not detect the address of my I2C pin. I was suggested to use an oscilloscope to check if I was getting readings. However, it appears that the jumper wire had a loose connection and was able to read the address after minor modifications.
